#### DPLOY EXAMPLE

example to deploy on ftp server

-   create file: dploy.yaml
-   deploy command : dploy alias


    deploy_alias:
        host: "ftp.domain"
        user: "use"
        pass: "pwd"
        path:
            local: "/"
            remote: "/folder/"
    exclude: [".vscode/**/*.{xml,json,abc}",".idea/**/*.{xml,json,abc}",".gitignore","dploy.yaml","_local/**/*.{html,css,sql,xml,md,php,html,js,json,htaccess}"]
